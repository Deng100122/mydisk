简易网盘 - 使用说明

项目概述

这是一个简洁美观的静态网盘网站，支持文件下载、链接复制和文件搜索功能。网站采用响应式设计，同时支持深色和浅色模式，能够根据系统主题自动调整。

功能特点

  • 文件下载：点击文件旁的下载按钮即可下载文件
  • 复制链接：点击链接按钮可复制文件的分享链接
  • 文件搜索：通过搜索框可以快速查找所需文件
  • 主题切换：支持深色和浅色模式，可手动切换或跟随系统设置
  • 响应式设计：适配各种设备屏幕尺寸

项目结构

  • index.html - 主页面
  • css/style.css - 样式文件
  • js/script.js - JavaScript功能实现
  • files/ - 存放供用户下载的文件
  • img/ - 存放网站使用的图片资源

使用方法

  1. 将您要分享的文件放入 files/ 文件夹中
  2. 修改 js/script.js 中的 fileData 数组，添加您的文件信息
  3. 部署网站到您的服务器或静态网站托管服务

注意：这是一个纯前端静态网站，没有后端服务。在实际应用中，您可能需要添加后端服务来动态获取文件列表和处理文件上传等功能。

自定义修改

您可以根据需要修改以下内容：

  • 在 css/style.css 中修改颜色变量，自定义网站主题
  • 在 js/script.js 中修改 fileData 数组，更新文件列表
  • 调整 index.html 中的标题和页脚信息

浏览器兼容性

本网站兼容所有现代浏览器，包括但不限于：

  • Google Chrome
  • Mozilla Firefox
  • Microsoft Edge
  • Safari
